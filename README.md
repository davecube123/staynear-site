# StayNear

StayNear is a simple travel app that helps users **find hotels near popular landmarks**.  
The MVP version focuses on **Siargao, Philippines**, with future expansion to other tourist destinations.

---

## Features
- Search for hotels around landmarks (using preset chips and autocomplete).
- Adjustable radius (1 km, 5 km, 20 km).
- Map confirmation of the selected landmark and search radius.
- Results with hotel details and Booking.com deep links.
- Monetization via the **Booking.com Affiliate Program**.

---

## Purpose
Travelers often want to stay *near* a specific spot (surf break, tourist landmark, etc.).  
StayNear makes this quick and simple: choose a landmark → set a radius → see hotels nearby.
